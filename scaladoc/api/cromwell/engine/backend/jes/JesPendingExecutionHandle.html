case class JesPendingExecutionHandle(backendCall: JesBackendCall, jesOutputs: Seq[JesOutput], run: Run, previousStatus: Option[RunStatus]) extends ExecutionHandle with Product with Serializable

Representing a running JES execution, instances of this class are never Done and it is never okay to ask them for results.
